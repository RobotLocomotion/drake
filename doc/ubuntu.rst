.. _build_from_source_ubuntu:

******
Ubuntu
******

Prerequisite setup is automated, simply run::

    sudo ./setup/ubuntu/install_prereqs.sh

On Ubuntu 18.04 (Bionic), you may need to respond to interactive prompts to
confirm that you agree to add the Drake `apt` repository
(``https://drake-apt.csail.mit.edu/bionic/``) to your system for certain
development packages.
