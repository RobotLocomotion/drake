pathlcp(M,q,l,u,z,A,b,t,mu)

Solve the standard linear complementarity problem using PATH: z >= 0, Mz + q >= 0, z'*(Mz + q) = 0

Required input: M(n,n) - matrix q(n) - vector

Output: z(n) - solution mu(m) - multipliers (if polyhedral constraints are present)

Optional input: l(n) - lower bounds default: zero u(n) - upper bounds default: infinity z(n) - starting point default: zero A(m,n) - polyhedral constraint matrix default: empty b(m) - polyhedral right-hand side default: empty t(m) - type of polyhedral constraint default: 1 < 0: less than or equal 0: equation > 0: greater than or equal mu(m) - starting value for multipliers default: zero

The optional lower and upper bounds are used to define a linear mixed complementarity problem (box constrained variational inequality). l <= z <= u where l_i < z_i < u_i => (Mz + q)_i = 0 l_i = z => (Mz + q)_i >= 0 u_i = z => (Mz + q)_i <= 0

The optional constraints are used to define a polyhedrally constrained variational inequality. These are transformed internally to a standard mixed complementarity problem. The polyhedral constraints are of the form Ax ? b where ? can be <=, =, or >= depending on the type specified for each constraint.

Return values:
z
mu
