CELL ODE simple function which abstracts the details of integrating a collection of vectors and/or matrices through the ODE solvers.

Take functions of the form: Adot = ODEFUN(t,A) where A is a 1xn cell array. The function returns tout,Aout where Aout{n}{i} is the value of A{i} at time tout(n).
