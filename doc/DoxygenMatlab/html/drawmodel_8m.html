drawmodel display system model or animation drawmodel(model) will display a system model, and drawmodel(model,delta_t,jnt_vals) will show an animation.

Both calls create a text file in the current directory, and ty to display it by calling the program anim8 (1st choice) or SceneViewer (2nd choice). The first call creates a file called data_km.iv that depicts the model's appearance and kinematics. The second call creates a file called data_mv.iv that depicts the motion described by jnt_vals. Arguments: model is a system-model data structure; jnt_vals is a 2-dimensional array with as many rows as data points, and as many columns as joints in the system model; and delta_t is the time step between consecutive data points.
