function append ( obj  ,
trajAtEnd   
)

Append a PPTrajectory to this one, creating a new trajectory that starts where this object starts and ends where the given trajectory ends.

This will throw an error if the trajectory to append does not start where the first trajectory ends. This is useful if you did a bunch of peicewise simulation and now want to combine them into one object.

Parameters:
trajAtEndtrajectory to append
Return values:
newtrajnew PPTrajectory object that is the combination of both trajectories
