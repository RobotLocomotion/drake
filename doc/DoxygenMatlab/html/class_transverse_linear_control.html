tau = computeTau(x0traj,ztraj,x,do_plot)

Computes the nearest time index tau with respect to transversal surfaces defined by z(tau).

Inputs:

  • x0traj: the state trajectory (trajectory object)
  • ztraj: the definition of transversal surfaces (trajectory object)
  • the current state to query Output:
  • tau: a time index of xtraj with x in the surface defined by by ztraj(tau)
  • zeroval: ztraj(tau)'*x, should be zero if successful
  • exitflag: exit flag returned from Matlab fzero. irm@mit.edu
    Return values:
    tau
    zeroval
    exitflag
