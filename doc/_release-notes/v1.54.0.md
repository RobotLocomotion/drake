# Breaking changes since v1.53.0

<!-- <relnotes for breaking-changes go here> -->

* Add drake::systems::PeriodicEventData::operator<=> ([#24559][_#24559])
  * This won't be breaking if users are capturing maps using PeriodicEventData
    using `auto`.

Refer to our [Drake Stability Guidelines](/stable.html) for our policy
on API changes.

# Notes


This release provides
[pre-compiled binaries](https://github.com/RobotLocomotion/drake/releases/tag/v1.54.0).
See [Installation](/installation.html) for instructions on how to use them.

Drake binary releases incorporate a pre-compiled version of
[SNOPT](https://ccom.ucsd.edu/~optimizers/solvers/snopt/) as part of the
[Mathematical Program toolbox](/group__solvers.html). Thanks to
Philip E. Gill and Elizabeth Wong for their kind support.
