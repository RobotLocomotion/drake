# Notes


This release provides
[pre-compiled binaries](https://github.com/RobotLocomotion/drake/releases/tag/v1.53.0).
See [Installation](/installation.html) for instructions on how to use them.

Drake binary releases incorporate a pre-compiled version of
[SNOPT](https://ccom.ucsd.edu/~optimizers/solvers/snopt/) as part of the
[Mathematical Program toolbox](/doxygen_cxx/group__solvers.html). Thanks to
Philip E. Gill and Elizabeth Wong for their kind support.
