# Drake Tutorials

## Tutorial on modeling dynamical systems
Drake provides a framework for modeling dynamical systems and combining them
 into block diagrams.  For an introduction to the systems framework, launch 
 the Jupyter notebook
```
bazel run //tutorials:dynamical_systems
```

## Tutorials on solving mathematical programs
Drake wraps formulating and solving optimization programs into a class `MathematicalProgram`. To use this class, you could refer to the tutorial by launching the Jupyter notebook
```
bazel run //tutorials:mathematical_program
```
